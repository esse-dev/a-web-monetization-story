I run a blog about my adventures with Chicken -- my dog!

Recently, I came across something called Web Monetization. It’s a new way for online creators to monetize their sites and for website visitors to support creators. It seeks to promote a more flexible, inclusive and privacy-respecting way of transferring money.

I love photographing and writing about my endeavours and funny mishaps with Chicken. It’s my passion, but it’s also a lot of time and effort, and I am always looking for ways to better support myself financially.

I decided to give my readers an option to support me through Web Monetization.

Essentially, Web Monetization lets my readers send me micropayments based on the time they spend on my site.

All I had to do to enable monetization on my site was add the Web Monetization meta tag.

It’s easy! Drag the <meta> tag into the <head> tag in the html on the right. 

<html lang="en">
    <head>
        

    </head>
    <body>
        <script>
        </script>
    </body>
</html>

Including this one line of code allows visitors of my website to stream money to me via a Web Monetization Provider. 

<html lang="en">
    <head>
        <meta name="monetization" content="MY_PAYMENT_POINTER">
    </head>
    <body>
        <script>
        </script>
    </body>
</html>

I know if a visitor is using Web Monetization if the document.monetization object exists. Let’s add a check in the <script> tag to detect the object! 

<html lang="en">
    <head>
        <meta name="monetization" content="MY_PAYMENT_POINTER">
    </head>
    <body>
        <script>
            

        </script>
    </body>
</html>

I can use this check to customize the website experience if a visitor is using Web Monetization. 

<html lang="en">
    <head>
        <meta name="monetization" content="MY_PAYMENT_POINTER">
    </head>
    <body>
        <script>
            if (document.monetization) {

            }
        </script>
    </body>
</html>

If the document.monetization object exists, I can check if a user has sent money by listening for the monetizationprogress event.

Let’s listen to the event in our script! 

<html lang="en">
    <head>
        <meta name="monetization" content="MY_PAYMENT_POINTER">
    </head>
    <body>
        <script>
            if (document.monetization) {
                

            }
        </script>
    </body>
</html>

Now, I can show my appreciation to those supporting me via Web Monetization by adding some code to enhance their experience. 

<html lang="en">
    <head>
        <meta name="monetization" content="MY_PAYMENT_POINTER">
    </head>
    <body>
        <script>
            if (document.monetization) {
                document.monetization.addEventListener('monetizationprogress', event => {

                });
            }
        </script>
    </body>
</html>

I’d like to show a “thank you” note as a token of my appreciation to those sending payment to me.

First, let's add a <div> with our note of appreciation and have it hidden by default. 

<html lang="en">
    <head>
        <meta name="monetization" content="MY_PAYMENT_POINTER">
    </head>
    <body>
        

        <script>
            if (document.monetization) {
                document.monetization.addEventListener('monetizationprogress', event => {

                });
            }
        </script>
    </body>
</html>

If a visitor has sent payment, we want to let them know it's appreciated!

Let’s show the <div> containing the note in the monetizationprogress event handler. 

<html lang="en">
    <head>
        <meta name="monetization" content="MY_PAYMENT_POINTER">
    </head>
    <body>
        <div id="thank-you-note" style="display: none;">
            Thank you for supporting me via Web Monetization <3
        </div>
        <script>
            if (document.monetization) {
                document.monetization.addEventListener('monetizationprogress', event => {
                    

                });
            }
        </script>
    </body>
</html>

That’s the gist of how I’ve used Web Monetization!

Since adding it to my blog and letting my visitors know, those who’ve opted in now support me with micropayments as they read about my adventures with Chicken.

It’s also great to chat with other creators using Web Monetization and engage in the community around it. 

<html lang="en">
    <head>
        <meta name="monetization" content="MY_PAYMENT_POINTER">
    </head>
    <body>
        <div id="thank-you-note" style="display: none;">
            Thank you for supporting me via Web Monetization <3
        </div>
        <script>
            if (document.monetization) {
                document.monetization.addEventListener('monetizationprogress', event => {
                    document.getElementById('thank-you-note').style.display = 'block';
                });
            }
        </script>
    </body>
</html>
